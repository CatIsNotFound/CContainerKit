# CContainerKit

基于 C11 标准的 C 语言容器集合。

## 特性

- 支持 C11 标准的 C 语言扩展库。
- 提供了各种容器的实现，如静态数组、动态数组、栈、队列等。
- 支持任意类型变量的存储和操作。

## 快速开始

访问 [Quick Start 页面](https://catisnotfound.github.io/CContainerKit-Wiki/zh/quickstart)以快速上手此库。

|----------|----------|--------------|
| CVariant | 70%      | 任意类型变量 |
| CArray   | 90%      | 静态数组     |
